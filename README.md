# Online Book Store

This is a simple frontend Online Book Store web application created for students to browse engineering and academic books. Books are organized department-wise to make searching easy and user-friendly.

## Features
- Department-wise book categories (CSE, ECE, EEE, CIVIL)
- Simple and clean user interface
- Easy navigation between pages
- Separate pages for Login, Registration, Catalogue, and Cart

## Technologies Used
- HTML
- CSS

## How to Run
1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` in any web browser.

## Purpose
This project is created for learning frontend web development concepts and practicing basic project structure.
